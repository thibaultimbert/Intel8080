Game is running in demo mode by default.

Single Player Mode :

Press (C) to start the single player mode
Press (S) to start.

MultiPlayer Mode :

Press (C) two times to get into multiplayer mode.
Press (S) to start.

Keyboard shortcuts :

P : Pause
R : Reset
LEFT : Move left
RIGHT : Move right
SPACE : Shoot
